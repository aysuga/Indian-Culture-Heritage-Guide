# Indian Culture & Heritage Guide

## Project Description
The Indian Culture & Heritage Guide is a frontend web application developed using HTML, CSS, and JavaScript.  
It provides state and union territory wise information about Indian culture, food, festivals, dance forms, and famous places.  
The project focuses on interactive UI design, dynamic content rendering, and responsive user experience.

## Technologies Used
- HTML5 – Structure and semantic layout
- CSS3 – Styling, responsiveness, dark mode, and animations
- JavaScript (ES6) – Logic, interactivity, DOM manipulation, and API handling
- VS Code Live Server – Used for local development and testing

## Steps to Run the Project
1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Right-click on `index.html` and select **Open with Live Server**.
5. The project will run in your default web browser.
